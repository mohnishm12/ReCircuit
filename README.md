# ReCircuit

Production Next.js website for `recircuit.online`, positioned as an honest early-stage climate AI startup founded by Mohnish M.

## Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS v4
- React Three Fiber, Drei, Three.js
- Lenis, Framer Motion, GSAP
- Lucide, shadcn-style primitives

## Commands

```bash
npm install
npm run dev
npm run build
npm run verify:visual
```
